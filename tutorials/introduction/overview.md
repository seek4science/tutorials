# What is FAIRDOM-SEEK?

FAIRDOM-SEEK is a web-based platform for sharing heterogeneous scientific research datasets, models, simulations, processes, and research outcomes.

## Key Features

- **FAIR Data Management** - Supports Findable, Accessible, Interoperable, Reusable principles
- **ISA Framework** - Organizes research using Investigation-Study-Assay structure
- **Flexible Assets** - Store data files, models, SOPs, publications, and more
- **Collaboration** - Share with project members or make data public
- **Versioning** - Track changes and create citable snapshots

## Who uses SEEK?

SEEK is used by research communities worldwide, including:

- Systems biology researchers
- Life science consortia
- Research data managers
- Core facilities

For an up to date list of SEEK instances, please refer to the [FAIRDOM-SEEK In Use](https://fair-dom.org/fairdom-in-use/).
